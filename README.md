# Smart Hostel / PG Management System

A web and mobile-based management system built using Zoho Creator to automate hostel and PG operations such as tenant management, room allocation, rent tracking, and complaint handling.

---

## 🚀 Features

- Tenant Management
  - Add, update, and remove tenants
  - Track tenant status (Active/Vacated)

- Room Management
  - Define room capacity
  - Auto-check availability
  - Auto-update occupancy

- Rent & Payment Tracking
  - Monthly rent records
  - Payment mode tracking
  - Pending and paid status
  - Due date calculation

- Complaint Management
  - Tenants can raise complaints
  - Admin can update complaint status

- Dashboard & Reports
  - Total tenants
  - Monthly revenue
  - Pending rent
  - Open complaints

- Automation
  - Rent reminder notifications
  - Auto room release on tenant exit
  - Auto occupancy update

---

## 🏗 Tech Stack

- Platform: Zoho Creator
- Database: Zoho Creator built-in database
- Automation: Deluge scripting
- UI: Zoho Creator Pages & Forms

---

## 📊 Modules

1. Tenants
2. Rooms
3. Payments
4. Complaints
5. Users (Roles)

---

## 🔐 User Roles

- Admin: Full access
- Staff: Limited management access
- Tenant: Complaint submission & payment view

---

## 🔄 Workflows

- On Tenant Add:
  - Check room capacity
  - Assign room if available

- On Payment:
  - Update rent status
  - Update dashboard metrics

- On Complaint:
  - Notify admin
  - Track resolution status
